HTML标签小记

January 4, 2022

记录一下HTML常用的标签

a 标签的用法

属性

  • href

    规定链接的目标 URL

    1. 可以是网址，
    2. 文件路径，
    3. 以及伪协议（Javascript:代码）
    4. id href=#XXX

    特别注意：

    1. 如果href的value为""，那么每次跳转都会刷新页面
    2. 如果href的value为"#"，那么每次跳转都会自动拉回到顶部
    3. 如果href的value为"javascript:;"，即不刷新也不跳转到顶部，啥也不干

  • target

    规定在何处打开目标 URL。仅在 href 属性存在时使用。

    1. _blank ：新窗口打开
    2. _parent ：在父窗口中打开链接
    3. _self : 默认，当前页面跳转。
    4. _top : 在当前窗体打开链接，并替换当前的整个窗体
    5. XXX ：新窗口打开, 且每次新打开的窗口更新为XXX

table 标签的用法

属性

  • thead
  • tbody
  • tfoot
  • tr
  • td
  • th
<table>
    <thead>
        <tr>
            <th></th>
            <th>数学</th>
            <th>语文</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <th>小明</th>
            <td>90</td>
            <td>80</td>
        </tr>
        <tr>
            <th>小王</th>
            <td>80</td>
            <td>90</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th>总分</th>
            <td>170</td>
            <td>170</td>
        </tr>
    </tfoot>
</table>

总分    170     170

img 标签的用法

作用

发出一张get请求，展示一张图片

属性

  • src

    图片的路径

  • alt

    图片加载失败后显示的内容

  • height weight

    图片的高度和宽度，调整一项，另一项会自动匹配，所以最好不要高度和宽度都自己调整

响应式

max-width:100% 可以让图片占满整个屏幕的宽度
